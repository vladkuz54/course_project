# course_project
 
Інструкція зі запуску:

1. Встановити Python версії 3.11 і вище

2. Зайти в папку з проєктом та відкрити два вікна термінала

3. У першому терміналі перейти у папку client, у другому flask-server

4. У терміналі з папкою client написати npm i, а після завершення встановлення npm run build

5. У терміналі з папкою flask-server написати python -m venv venv та після встанолвення vevn\scripts\activate. Після активуванння venv'у написати pip install flask

6. Після завершення всіх встановлень у терміналі з папкою flask-server написати python server.py

7. Перейти на http://localhost:5000
